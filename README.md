# 🛠 Redmine User Unlocker

GUI-утиліта для швидкого масового розблокування користувачів у системі Redmine за списком логінів.

## 📂 Структура проєкту

У цьому репозиторії зберігаються дві версії скрипта:

* **`main.py` (Стара версія):** Базовий функціонал. Завантажує API-ключ безпечно через `.env`. Працює як простий інструмент без зайвого навантаження.
* **`unlocker.py` (Нова версія):** Покращений функціонал. Додано візуальний прогрес-бар, перевірку на дублікати, локальне логування всіх дій у файл `unlock_history.log` та можливість експортувати звіт у `.txt`.

## 📊 Архітектурна схема роботи (Логіка утиліти)

```mermaid
graph TD
    subgraph "1. Підготовка даних (Інтерфейс)"
        Start([Користувач вставляє список логінів]) --> Validate[Валідація: Видалення дублікатів та порожніх рядків]
    end

    subgraph "2. Основний цикл обробки (Redmine API)"
        Validate --> Loop{Цикл по унікальних логінах}
        
        Loop --> CheckAPI[Пошук користувача: login + status=3]
        CheckAPI --> Decision{Знайдено заблокованого?}
        
        Decision -- "Ні (Не існує / Активний)" --> Skip[Пропуск: Запис у лог]
        Decision -- "Так (Заблокований)" --> Unlock[API Update: Зміна status=1]
        
        Unlock --> UIUpdate[Оновлення прогрес-бару та UI]
        Skip --> UIUpdate
        
        UIUpdate --> CancelCheck{Натиснуто 'Відміна'?}
        CancelCheck -- "Ні" --> Loop
    end

    subgraph "3. Завершення"
        CancelCheck -- "Так" --> Finish
        Loop -. "Список вичерпано" .-> Finish([Кінець: Звуковий сигнал та Статистика])
        Finish --> Export[Експорт .txt звіту]
    end
```

## ⚙️ Встановлення

1. Склонуйте репозиторій.
2. Встановіть залежності: `pip install -r requirements.txt`
3. Запустіть потрібну вам версію скрипта.
